# QC Images

### For Masks

`generate_qc_images.py` creates QC images of the requested slice numbers for input
background and mask images. Ranges are an acceptable input to -x, -y, and -z
N.B. Slice indexing is 0-based (e.g., 0-255)

Required package:
* nibabel
